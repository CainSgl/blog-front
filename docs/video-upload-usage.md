# 视频/动图上传功能使用说明

## 功能概述

参考音乐播放器组件的实现方式，新增了视频和动图（GIF）上传功能。用户可以在 Markdown 编辑器中轻松插入视频或动图内容。

## 使用方法

### 1. 在编辑器工具栏中插入

在 Markdown 编辑器的工具栏中，找到"插入视频/动图"按钮（播放图标），点击后：

1. 选择视频文件（支持常见视频格式：mp4, webm, ogg 等）或 GIF 动图
2. 文件会自动上传到服务器
3. 上传成功后，会在编辑器中插入对应的 Markdown 语法

### 2. 右键菜单插入

在编辑器中右键点击，选择"插入视频/动图"选项，操作方式同上。

### 3. Markdown 语法

插入的视频/动图使用以下语法格式：

```markdown
:::video 文件ID 视频名称.mp4
:::
```

或者对于 GIF：

```markdown
:::video 文件ID 动图名称.gif
:::
```

## 功能特性

### 视频播放器特性

- **完整的视频控制**：播放、暂停、进度条、音量控制
- **倍速播放**：支持 0.5x、0.75x、1.0x、1.25x、1.5x、2.0x 多种播放速度
- **响应式设计**：自动适应不同屏幕尺寸
- **暗黑模式支持**：完美适配明暗主题切换

### GIF 动图特性

- **自动播放**：GIF 会自动循环播放
- **静音播放**：GIF 默认静音
- **简化控制**：GIF 不显示复杂的控制面板，保持简洁

### 视觉设计

- 卡片式设计，带有渐变背景和阴影效果
- 红色主题图标，与音乐播放器的蓝色形成区分
- 悬停动画效果，提升交互体验
- 文件类型标识（GIF/视频）

## 技术实现

### 组件结构

```
src/components/md/common/VideoPlayer.vue  # 视频播放器组件
src/components/md/MarkdownEditor.vue      # 编辑器（添加插入功能）
src/components/md/MarkdownPreview.vue     # 预览组件（渲染视频）
src/plugins/md-tip-info-extens.js         # Markdown 扩展（解析语法）
```

### 实现原理

1. **上传流程**：使用 `uploadFile` 工具类上传文件到服务器，获取文件 ID
2. **语法解析**：通过自定义 Markdown 扩展解析 `:::video` 语法
3. **组件渲染**：在预览时将占位符替换为 Vue 视频播放器组件
4. **文件访问**：通过 API 端点 `/file?f={fileId}` 访问上传的文件

## 与音乐播放器的对比

| 特性 | 音乐播放器 | 视频播放器 |
|------|-----------|-----------|
| 主题色 | 蓝色 | 红色 |
| 图标 | 音乐图标 | 播放圆圈图标 |
| 文件类型 | 音频文件 | 视频文件 + GIF |
| 自动播放 | 需确认 | GIF 自动播放 |
| 控制面板 | 始终显示 | 视频显示，GIF 隐藏 |

## 注意事项

1. **文件大小限制**：建议视频文件不超过 100MB，以确保良好的加载体验
2. **格式兼容性**：推荐使用 MP4 格式，兼容性最好
3. **GIF 优化**：大型 GIF 建议转换为视频格式以减小文件大小
4. **浏览器支持**：需要现代浏览器支持 HTML5 video 标签

## 示例

### 插入普通视频

```markdown
:::video abc123 我的演示视频.mp4
:::
```

### 插入 GIF 动图

```markdown
:::video def456 搞笑动图.gif
:::
```
