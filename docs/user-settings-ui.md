# 用户设置界面使用说明

## 功能概述

在用户个人主页添加了一个完整的设置管理界面，用户可以方便地查看和编辑所有自定义设置。

## 访问方式

1. 登录后访问自己的个人主页 `/space/{userId}`
2. 点击头像下方的 **"个人设置"** 按钮
3. 在弹出的对话框中编辑设置

## 设置分类

### 1. 音乐播放器
- **默认音量**: 拖动滑块设置 0-100%
- **默认播放速度**: 选择 0.5x - 2.0x
- **播放确认**: 开启后音乐自动播放，不再弹出确认对话框

### 2. 外观
- **主题模式**: 浅色 / 深色 / 跟随系统
- **语言**: 简体中文 / 繁體中文 / English

### 3. 通知
- 评论通知
- 点赞通知
- 关注通知
- 系统通知

### 4. 隐私
- 显示邮箱
- 显示手机号
- 允许被搜索
- 显示在线状态

### 5. 高级
- **原始设置数据**: 直接编辑 JSON 格式的设置
- **格式化**: 格式化 JSON 数据
- **恢复默认**: 恢复所有设置为默认值
- **清除所有设置**: 删除本地和服务器的所有设置

### 6. 云同步 ⭐ 新功能
- **同步到云端**: 将本地设置上传到服务器
- **从云端同步**: 从服务器下载设置到本地
- **云端状态**: 查看本地和云端设置数量、最后同步时间
- **云端预览**: 查看云端保存的设置内容

## 特性

✅ **实时预览**: 修改设置后立即在界面上反映
✅ **本地保存**: 点击保存后保存到本地缓存（不自动同步云端）
✅ **手动云同步**: 用户完全控制何时同步到云端
✅ **多设备支持**: 通过云同步在多设备间共享设置
✅ **JSON 编辑**: 高级用户可以直接编辑 JSON 数据
✅ **数据验证**: 保存前会验证 JSON 格式
✅ **云端预览**: 查看云端保存的设置内容

## 使用场景

### 场景 1: 调整音乐播放器默认设置
1. 打开个人设置
2. 切换到"音乐播放器"标签
3. 调整音量和播放速度
4. 开启"不再提示"
5. 点击保存

### 场景 2: 切换主题
1. 打开个人设置
2. 切换到"外观"标签
3. 选择主题模式
4. 点击保存

### 场景 3: 批量编辑设置（高级）
1. 打开个人设置
2. 切换到"高级"标签
3. 直接编辑 JSON 数据
4. 点击"格式化"验证格式
5. 点击保存

### 场景 4: 多设备同步设置 ⭐
**在主设备上:**
1. 打开个人设置
2. 配置好所有设置
3. 点击保存
4. 切换到"云同步"标签
5. 点击"同步到云端"

**在新设备上:**
1. 登录账号
2. 打开个人设置
3. 切换到"云同步"标签
4. 点击"从云端同步"
5. 设置自动应用到本地

### 场景 5: 查看云端设置
1. 打开个人设置
2. 切换到"云同步"标签
3. 查看"云端状态"了解同步情况
4. 在"云端设置预览"中查看云端保存的内容

## 技术实现

- **组件**: `UserSettingsModal.vue`
- **位置**: `src/views/user/children/home/components/`
- **Store**: `useUserSettingStore`
- **API**: `/user/setting` (GET/PUT/DELETE)

## 注意事项

1. 只有访问自己的个人主页时才会显示"个人设置"按钮
2. 设置默认保存到本地缓存，**不会自动同步到云端**
3. 需要在多设备间共享设置时，请使用"云同步"功能
4. 退出登录后本地缓存会被清除（云端数据不受影响）
5. 高级设置中的 JSON 编辑需要注意格式正确性
6. 清除所有设置会同时删除本地和服务器数据，请谨慎操作
7. "同步到云端"会覆盖云端设置，"从云端同步"会覆盖本地设置
