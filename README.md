## 最新更新

### PWA 支持 - 安装到桌面 🚀

现在支持将网站安装到桌面，像原生应用一样使用！

**主要特性：**
- 📱 一键安装到桌面/主屏幕
- ⚡ 离线缓存，更快的加载速度
- 🎨 全屏体验，无浏览器地址栏
- 🔄 自动更新，无需手动操作
- 💾 智能缓存策略（API、图片、静态资源）

**如何安装：**
1. 访问网站后等待 3 秒，会弹出安装提示
2. 或点击浏览器地址栏的安装图标
3. 确认安装即可

详细说明请查看：[PWA 安装指南](docs/pwa-guide.md)

### 视频/动图上传功能 ✨

参考音乐播放器组件的实现方式，新增了视频和动图（GIF）上传功能！

**主要特性：**
- 📹 支持视频文件上传（MP4, WebM, OGG 等格式）
- 🎬 支持 GIF 动图上传
- 🎮 完整的播放控制（播放/暂停、进度条、音量、倍速）
- 🎨 精美的卡片式设计，支持明暗主题
- 🔄 GIF 自动循环播放
- 📱 响应式设计，适配各种屏幕

**使用方法：**
1. 在 Markdown 编辑器工具栏点击"插入视频/动图"按钮
2. 选择视频或 GIF 文件
3. 等待上传完成，自动插入到编辑器中

**Markdown 语法：**
```markdown
:::video 文件ID 视频名称.mp4
:::
```

详细使用说明请查看：[视频上传功能文档](docs/video-upload-usage.md)
