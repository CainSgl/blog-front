# PWA 安装指南

## 什么是 PWA？

PWA (Progressive Web App) 是一种可以像原生应用一样安装到设备上的网页应用。安装后，你可以：

- 📱 从桌面或主屏幕直接启动
- 🚀 更快的加载速度（离线缓存）
- 💾 离线访问部分内容
- 🎨 全屏体验，无浏览器地址栏
- 🔔 接收推送通知（未来功能）

## 如何安装？

### 方式一：自动提示（推荐）

1. 访问网站后，等待 3 秒
2. 底部会弹出"安装到桌面"的提示气泡
3. 点击"安装"按钮
4. 确认安装

### 方式二：浏览器菜单

#### Chrome / Edge (桌面)
1. 点击地址栏右侧的 ⊕ 或 💻 图标
2. 选择"安装 cainsgl的小站"
3. 确认安装

#### Chrome (Android)
1. 点击右上角菜单 (⋮)
2. 选择"添加到主屏幕"
3. 确认添加

#### Safari (iOS)
1. 点击底部分享按钮 (⬆️)
2. 滚动找到"添加到主屏幕"
3. 点击"添加"

## 功能特性

### 自定义标题栏

安装后的应用支持两种标题栏模式：

#### 1. Window Controls Overlay（推荐）
- 现代化的自定义标题栏
- 标题栏与应用内容融合
- 显示应用图标和页面标题
- 包含后退、前进、刷新按钮
- 支持拖拽移动窗口

#### 2. Standalone（标准模式）
- 系统原生标题栏
- 显示应用名称
- 标准的窗口控制按钮

浏览器会自动选择支持的最佳模式。Chrome/Edge 最新版支持 Window Controls Overlay。

### 主题色配置

- **主题色**: `#165dff` (Arco Design 主色)
- **背景色**: `#ffffff`
- 标题栏会自动适配明暗主题

### 桌面快捷方式

安装后，在桌面图标上右键（或长按），可以看到快捷方式菜单：

- **首页** - 快速查看最新文章和动态
- **知识库** - 直接进入知识库浏览
- **我的主页** - 查看个人主页和内容
- **消息中心** - 快速访问消息和通知

这些快捷方式让你可以直接跳转到常用页面，无需先打开首页。

### 分享目标

安装后，你可以从其他应用（如相册、文件管理器）直接分享内容到本应用：

1. 在其他应用中选择"分享"
2. 选择"cainsgl的小站"
3. 内容会自动打开在应用中

支持分享：图片、视频、音频文件和链接。

### 缓存策略

- **API 请求**: NetworkFirst - 优先网络，失败时使用缓存（24小时有效）
- **图片资源**: CacheFirst - 优先缓存，加快加载速度（30天有效）
- **JS/CSS**: StaleWhileRevalidate - 使用缓存同时后台更新（7天有效）

### 自动更新

应用会自动检测新版本并在后台更新，无需手动操作。

## 卸载方法

### Windows / macOS
1. 打开已安装的应用
2. 点击右上角菜单
3. 选择"卸载"

### Android
- 长按应用图标 → 卸载

### iOS
- 长按应用图标 → 删除应用

## 技术实现

本项目使用以下技术实现 PWA：

- `vite-plugin-pwa`: Vite PWA 插件
- `workbox`: Google 的 Service Worker 工具库
- Web App Manifest: 应用配置文件
- Service Worker: 离线缓存和后台同步

## 常见问题

**Q: 为什么没有看到安装提示？**
A: 可能原因：
- 已经安装过
- 7天内拒绝过安装
- 浏览器不支持 PWA
- 网站必须使用 HTTPS

**Q: 安装后如何更新？**
A: 应用会自动检测并更新，无需手动操作。

**Q: 离线能用吗？**
A: 部分功能可以离线使用，但需要先在线访问过一次。

**Q: 占用多少空间？**
A: 通常 10-50MB，取决于缓存的内容量。
